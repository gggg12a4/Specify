## 三、各区域说明

### 分享链接

| 元素 | 说明 |
|------|------|
| 链接文本 | 始终生成，不可编辑 |
| [复制] | 点击复制到剪贴板，toast 提示「已复制」 |
| 公开分享复选框 | 勾选后 App 出现在所有用户的「推荐 App」区域 |

### LLM 算力计费

| 选项 | 含义 | 使用者体验 |
|------|------|-----------|
| 使用者付费 | 使用者需要配置该平台的密钥或使用平台额度 | 首次使用时弹出密钥配置页 |
| 开发者付费 | 从开发者的平台余额中扣费 | 使用者无需任何模型配置 |

### MCP 服务凭证

仅当 App 配置了 MCP 服务时才显示此区域。

每个 MCP 服务提供以下选项：

| 选项 | 含义 | 使用者体验 |
|------|------|-----------|
| **让使用者自行填写凭证 (推荐)** | 使用者需要提供自己的凭证（默认选中） | 首次使用时必须填写（必填） |
| **共享我的凭证给所有人** | 使用开发者配置的凭证 | 使用者直接可用，显示 ✅ 已就绪 |

**没有 MCP 服务时**，此区域整块不显示。

### 配置摘要

根据当前选择**动态生成**文案，让开发者清楚看到使用者的体验。

生成规则：

| 条件 | 摘要项 |
|------|--------|
| 使用者付费 | "需要 {平台} 密钥或平台余额" |
| 开发者付费 | "无需配置模型，直接使用（你承担算力费用）" |
| MCP 共享凭证 | "可直接使用 {服务名} 工具（你的凭证）" |
| MCP 使用者填写 | "需要填写 {服务名} 凭证（连接使用者自己的 {服务名}）" |

## 四、交互逻辑

### 实时保存

分享设置的所有选项**修改后即时保存**（无需点保存按钮）：

| 操作 | 保存时机 |
|------|----------|
| 切换 LLM 计费选项 | 选中即保存 |
| 切换 MCP 凭证选项 | 选中即保存 |
| 勾选/取消公开分享 | 勾选即保存 |

### 配置摘要实时更新

切换任何选项后，底部配置摘要立即更新。

## 六、共享凭证风险提示弹窗（二次确认）

当开发者在「MCP 服务凭证」区域试图选中 `( ) 共享我的凭证给所有人` 时，**必须**强制弹出此风险提示弹窗。

```text
┌─────────────────────── ⚠️ 严重风险提示 ───────────────────────┐
│                                                          [✕]   │
│  您正在将 {服务名} 凭证公开给所有访问此链接的人。                  │
│  他们将能以您的身份读取、修改甚至删除您的私人数据。                │
│  平台不对由此造成的数据泄露负责。                                  │
│                                                                │
│  [ ] 我已了解风险                                                │
│                                                                │
│                                        [取消] [确认共享]       │
└────────────────────────────────────────────────────────────────┘
```
*(注：文案中的 `{服务名}` 需动态替换为对应的 MCP 工具名称，例如 notion、github)*

**交互规则：**
1. **防误触机制**：默认情况下，复选框未勾选，`[确认共享]` 按钮必须为**禁用（Disabled）**状态。
2. **强制确认**：只有当开发者手动勾选 `[x] 我已了解风险` 后，`[确认共享]` 按钮才恢复可点击状态。
3. **确认操作**：点击 `[确认共享]`，弹窗关闭，原界面的单选框成功切换为 `(●) 共享我的凭证给所有人`，并即时保存。
4. **取消操作**：点击 `[取消]` 或右上角 `[✕]`，弹窗关闭，原界面的单选框操作撤销，保持选中 `(●) 让使用者自行填写凭证 (推荐)`。
