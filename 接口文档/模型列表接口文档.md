

## 模型清单-编辑


**接口地址**:`/specify/specifysystem/modelList/edit`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": "",
  "createBy": "",
  "createTime": "",
  "updateBy": "",
  "updateTime": "",
  "modelGroupId": "",
  "name": "",
  "displayName": "",
  "type": "",
  "description": "",
  "inputPrice": 0,
  "outputPrice": 0,
  "visible": true,
  "status": 0,
  "sortOrder": 0,
  "delFlag": 0
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|modelList|模型清单|body|true|ModelList|ModelList|
|&emsp;&emsp;id|ID||false|string||
|&emsp;&emsp;createBy|创建人||false|string||
|&emsp;&emsp;createTime|创建时间||false|string(date-time)||
|&emsp;&emsp;updateBy|更新人||false|string||
|&emsp;&emsp;updateTime|更新时间||false|string(date-time)||
|&emsp;&emsp;modelGroupId|所属分组ID||true|string||
|&emsp;&emsp;name|API标识符||true|string||
|&emsp;&emsp;displayName|前端显示名||true|string||
|&emsp;&emsp;type|chat=主对话模型 / sub=专用子模型||true|string||
|&emsp;&emsp;description|模型描述||false|string||
|&emsp;&emsp;inputPrice|输入单价（元/百万token）||false|number||
|&emsp;&emsp;outputPrice|输出单价（元/百万token）||false|number||
|&emsp;&emsp;visible|是否在选择列表中展示（0=隐藏）||false|boolean||
|&emsp;&emsp;status|生命周期状态：1=正常，2=即将过期，3=失效||false|integer(int32)||
|&emsp;&emsp;sortOrder|排序权重（越小越靠前）||false|integer(int32)||
|&emsp;&emsp;delFlag|删除状态(0-正常,1-已删除)||false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultString|
|401|Unauthorized|ResultObject|


**响应状态码-200**:


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|success|成功标志|boolean||
|message|返回处理消息|string||
|code|返回代码|integer(int32)|integer(int32)|
|result|返回数据对象|string||
|timestamp|时间戳|integer(int64)|integer(int64)|


**响应示例**:
```javascript
{
	"success": true,
	"message": "",
	"code": 0,
	"result": "",
	"timestamp": 0
}
```


**响应状态码-401**:


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|success|成功标志|boolean||
|message|返回处理消息|string||
|code|返回代码|integer(int32)|integer(int32)|
|result|返回数据对象|object||
|timestamp|时间戳|integer(int64)|integer(int64)|


**响应示例**:
