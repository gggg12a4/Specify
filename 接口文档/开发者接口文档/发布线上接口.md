

## 发布到线上


**接口地址**:`/specify/specifyuser/app/publish`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:<p>将当前草稿（draft_config）发布为一个新的 app_versions 快照版本。
版本号自动递增（取当前 App 下最大 version_number + 1）。
发布成功后，App 的 liveVersionId 指向新版本，draftConfig 被清空。</p>
<p>请求体 JSON 字段：</p>
<ul>
<li><code>app_id</code> (String, 必填): App ID</li>
<li><code>force_upgrade</code> (Boolean, 可选, 默认 false): 是否强制升级</li>
<li><code>is_compatible</code> (Boolean, 可选, 默认 true): 是否兼容历史对话</li>
</ul>
<p>返回值：<code>{ &quot;versionId&quot;: &quot;...&quot;, &quot;versionNumber&quot;: 1 }</code></p>



**请求示例**:


```javascript
{
  "app_id": "",
  "force_upgrade": true,
  "is_compatible": true
}
```
