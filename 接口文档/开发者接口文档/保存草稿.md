

## 保存草稿


**接口地址**:`/specify/specifyuser/app/saveDraft`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:<p>将当前编辑配置保存为 App 草稿，不会对线上版本产生任何影响。</p>
<p>【重要契约】本接口为全量快照覆盖语义：每次保存必须回传完整配置，
未回传（缺省 / 为 null）的字段将被清空。请勿只回传被修改的字段（不做增量合并）。</p>
<p>外层 key 仅允许：app_id、model_group_id、system_prompt、selected_model、tool_snapshot
tool_snapshot 内层 key 仅允许：platform_tools、developer_mcp_tools、developer_agent_tools
platform_tools 的 ref_type 仅允许：platform_basetools、platform_agent_tools、platform_mcp_tools</p>



**请求示例**:


```javascript
{
  "app_id": "",
  "model_group_id": "",
  "system_prompt": "",
  "selected_model": "",
  "tool_snapshot": {
    "platform_tools": [
      {
        "ref_type": "",
        "ref_id": "",
        "enabled": true,
        "config": {}
      }
    ],
    "developer_mcp_tools": [
      {
        "id": "",
        "config": {
          "name": "",
          "url": "",
          "headers": {}
        },
        "enabled": true,
        "credential_sharing": ""
      }
    ],
    "developer_agent_tools": [
      {
        "id": "",
        "config": {
          "name": "",
          "description": "",
          "system_prompt": ""
        },
        "enabled": true,
        "subtools": [
          {
            "ref_type": "",
            "ref_id": "",
            "enabled": true,
            "config": {}
          }
        ]
      }
    ]
  }
}
```
