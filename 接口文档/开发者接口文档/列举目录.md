

## 列举目录


**接口地址**:`/specify/specifyfilesystem/api/v1/list`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "path": "",
  "recursive": false,
  "pattern": "",
  "file_types": [],
  "max_items": 100,
  "include_meta": true,
  "include_path": true,
  "item_type": "all"
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|listRequest|列举目录请求|body|true|ListRequest|ListRequest|
|&emsp;&emsp;path|目录路径||false|string||
|&emsp;&emsp;recursive|是否递归||false|boolean||
|&emsp;&emsp;pattern|文件名模式匹配||false|string||
|&emsp;&emsp;file_types|文件类型过滤||false|array|string|
|&emsp;&emsp;max_items|最大返回数量||false|integer(int32)||
|&emsp;&emsp;include_meta|是否包含元数据||false|boolean||
|&emsp;&emsp;include_path|是否包含路径||false|boolean||
|&emsp;&emsp;item_type|条目类型: files/dirs/all||false|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultObject|
|401|Unauthorized|ResultObject|


**响应状态码-200**:


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|success|成功标志|boolean||
|message|返回处理消息|string||
|code|返回代码|integer(int32)|integer(int32)|
|result|返回数据对象|object||
|timestamp|时间戳|integer(int64)|integer(int64)|


**响应示例**:
```javascript
{
	"success": true,
	"message": "",
	"code": 0,
	"result": {},
	"timestamp": 0
}
```
