

## 编辑页数据总接口


**接口地址**:`/specify/specifyuser/app/queryEditPageData`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:<p>获取 App 编辑页所需的全部数据，包括：</p>
<ul>
<li><strong>app</strong>: App 基本信息（id, name, description, model_group_id, billing_mode, is_public, share_code, live_version_id）</li>
<li><strong>model_group</strong>: 模型分组信息（id, name, display_name, status）</li>
<li><strong>config</strong>: 配置内容（system_prompt, selected_model, tool_snapshot），优先草稿其次已发布版本</li>
<li><strong>config_source</strong>: 配置来源（&quot;draft&quot;/&quot;live&quot;/null）</li>
<li><strong>has_draft</strong>: 是否有草稿</li>
<li><strong>draft_updated_at</strong>: 草稿最后更新时间（ISO 8601），无草稿时返回 null</li>
<li><strong>selected_model_status</strong>: 所选模型的生命周期状态</li>
<li><strong>versions</strong>: 版本历史列表</li>
<li><strong>models</strong>: 模型分组下的可用模型列表</li>
<li><strong>templates</strong>: 工具模板（platform_tools 字典 + createable 字典）</li>
</ul>



**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|appId|App ID|query|true|string||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|业务错误（HTTP 200，Result.code）：appId 缺失/空/格式错误/超长 → 400；App 不存在 → 404；无权限操作此 App → 403|ResultMapStringObject|
|401|Unauthorized|ResultObject|

**响应参数**:

|code|返回代码|integer(int32)|integer(int32)|
|result|返回数据对象|object||
|timestamp|时间戳|integer(int64)|integer(int64)|
