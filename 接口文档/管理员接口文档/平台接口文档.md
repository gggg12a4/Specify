

## 模型平台-编辑


**接口地址**:`/specify/specifysystem/modelGroups/edit`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": "",
  "createBy": "",
  "createTime": "",
  "updateBy": "",
  "updateTime": "",
  "name": "",
  "displayName": "",
  "description": "",
  "detail": "",
  "apiBase": "",
  "visible": true,
  "status": 0,
  "sortOrder": 0,
  "delFlag": 0
}
```


**请求参数**:


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|modelGroups|模型平台|body|true|ModelGroups|ModelGroups|

|&emsp;&emsp;createBy|创建人||false|string||
|&emsp;&emsp;createTime|创建时间||false|string(date-time)||
|&emsp;&emsp;updateBy|更新人||false|string||
|&emsp;&emsp;updateTime|更新时间||false|string(date-time)||
|&emsp;&emsp;name|分组标识||true|string||
|&emsp;&emsp;displayName|前端展示名称||true|string||
|&emsp;&emsp;description|分组简述||false|string||
|&emsp;&emsp;detail|分组详情||false|string||
|&emsp;&emsp;apiBase|默认API地址||true|string||
|&emsp;&emsp;visible|是否在选择列表中展示（0=隐藏）||false|boolean||
|&emsp;&emsp;status|生命周期状态：1=正常，2=即将过期，3=失效||false|integer(int32)||
|&emsp;&emsp;sortOrder|排序权重（越小越靠前）||false|integer(int32)||
|&emsp;&emsp;delFlag|删除状态(0-正常,1-已删除)||false|integer(int32)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultString|
